# ReactProject — Letterboxd Lite Rebuild

## Original Website
https://letterboxd.com

## Project Description
This project is a “lite rebuild” of Letterboxd using React.  
The goal is to recreate the core layout and user flow of the site using a
component-based React structure, with placeholder content for future features.

## How to Run Locally
1. Clone the repository
2. Navigate into the project folder
3. Install dependencies and start the dev server:

```bash
npm install
npm run dev# reactproject
